# 🏡 RealEstateApp – Real Estate Website  

**RealEstateApp** is a modern web platform for searching, selling, and renting properties. It utilizes **Vue 3 + Nuxt 3** for the frontend and **Node.js + Express + MongoDB** for the backend, with support for **Firebase authentication**, including Google login.  

---

## ⚒️ App Features  

✅ Search for properties by address  
✅ Filtering results and pagination  
✅ Detailed property view with an image gallery  
✅ User registration and Google/Firebase login 🔥  
✅ Admin panel for managing listings  
✅ Property scraping from external websites  
✅ Adding properties to favorites ❤️  

---

## ⚙️ Technologies  

### **Frontend (Vue 3 + Nuxt 3 + Tailwind CSS)**  
- Vue 3 (Composition API)  
- Nuxt 3 (SSR, SSG, Dynamic Routing)  
- Tailwind CSS (modern design)  
- Vue Router (navigation)  
- VueFire (Firebase authentication)  
- PhotoSwipe (property image gallery)  

### **Backend (Node.js + Express + MongoDB)**  
- Express.js (REST API)  
- MongoDB + Mongoose (data layer)  
- Firebase Auth (user authentication)  
- Scraping with Cheerio.js
